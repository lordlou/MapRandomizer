Race mode

When set to "Yes", spoiler data is initially locked. The creator of the seed can unlock the spoiler data, at which point it will become viewable to everyone, and the timestamp when the spoiler data was unlocked will be shown on seed page.

The seed creator will need to use the same device and browser to unlock the spoiler data as what they used to create it (as the ability to unlock the data depends on a small random token in browser-local storage).
