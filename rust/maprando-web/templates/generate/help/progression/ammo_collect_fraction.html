Ammo collect fraction

This determines what fraction of logically collectible ammo packs the player may be required to collect. This affects Missiles, Supers, and Power Bombs. It should be set to a value between 0.5 and 1.0. For each type of ammo, the number of packs that are logically collectible is multiplied by this fraction, then rounded to the nearest integer, to determine the amount of packs assumed by the logic to have been collected.

Setting this fraction to 1.0 means that everything available is assumed to be collected, which means that in some scenarios the player could be required to sweep the world to collect every available pack. The default value of 0.7 ensures that the player will have more flexibility.
