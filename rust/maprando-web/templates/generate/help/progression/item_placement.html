Item placement

This setting affect how the randomizer selects locations to place key items in on each step, among all the possible obtainable item locations as determined by "Skill assumptions".

  • Neutral: Key items are placed in random locations (among the possible locations).

  • Forced: Key items tend to be placed in locations requiring difficult techniques to reach.

  • Local: Key items tend to be placed in locations requiring difficult techniques to reach, and in case of a tie in difficulty, preference is given to a location in the same area as a key item on the previous step.
