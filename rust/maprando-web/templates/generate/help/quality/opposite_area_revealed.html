Opposite area connections revealed by map

If enabled, activating a map stations causes area-transition markers (arrows or letters) to be revealed in neighboring areas, showing connections leading back to the area where the map station was activated.

This option can make it easier for players to visualize how areas connect together.

Note that regardless of this setting, map stations always reveal area-transition markers in the area where the map station was activated, showing outgoing connections to other areas. Also, crossing an area transition always reveals both sides of the connection.
