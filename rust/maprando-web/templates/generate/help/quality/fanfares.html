Item fanfares

Controls the time it takes for the dialogue box that pops up upon obtaining an item to close.

  • Vanilla: Exactly as in the original game. Dialogue box lasts for 6 seconds before it can be closed.
  • Trimmed: The fanfare plays as usual, but the box can be closed as soon as the fanfare starts to fade out (4 seconds).
  • Off: No fanfare is played. Instead, a short sound effect which depends on the item is played. Box lasts 16 frames (there are 60 frames in a second).
